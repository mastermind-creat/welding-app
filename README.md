## âš™ï¸ Development Setup

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone git@github.com:mastermind-creat/welding-app.git
   cd welding-app
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
